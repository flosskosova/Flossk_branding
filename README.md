# FLOSSK Brand Guidelines

**Free/Libre Open Source Software Kosova** is a non-governmental organization based in Prishtina, Kosovo, established in 2009 to support, promote, and develop free and open source software and its community.

> The authoritative source for all brand guidelines is [`flossk_brandbook.pdf`](./flossk_brandbook.pdf) (in Albanian).

---

## Logo

FLOSSK has five logo variants. The **primary logo** should be used in all communications by default.

| Variant | Contents | Use Case | Minimum Size |
|---------|----------|----------|--------------|
| [Primary](./logos/primary/) | Acronym + crow symbol | Default — all communications | 20mm / 150px |
| [Secondary](./logos/secondary/) | Symbol + acronym + full name | Full brand representation | 20mm / 150px |
| [Tertiary](./logos/tertiary/) | Symbol + full name in circular form | Rare cases — badge use | 20mm / 150px |
| [Symbol](./logos/symbol/) | Crow symbol standalone | Icon contexts | 10mm / 50px |
| [Stema](./logos/stema/) | Redesigned badge with Alfabeti lettering | Ceremonial / special use | 20mm / 150px |

Each variant folder contains files named by color: `black.svg`, `white.svg`, `yellow.svg` (symbol only). PNG and PDF versions are included alongside SVGs. The `symbol/` folder also includes `sorra.svg` — the standalone crow illustration.

### Logo Usage Rules

Always use the original logo without modification. The following are not permitted:

- Stretching or deforming the logo
- Changing the size or position of the symbol relative to the logotype
- Cropping or clipping any part of the logo
- Adding drop shadows or visual effects
- Using all-capital letters
- Outlining the logo

## Colors

FLOSSK's brand palette is black, white, and yellow. Yellow is used as an accent color.

| Swatch | Name | Hex | CMYK |
|--------|------|-----|------|
| ![#000000](https://placehold.co/24x24/000000/000000.png) | Black | `#000000` | C:0 M:0 Y:0 K:100 |
| ![#ffffff](https://placehold.co/24x24/ffffff/ffffff.png) | White | `#ffffff` | C:0 M:0 Y:0 K:0 |
| ![#fff200](https://placehold.co/24x24/fff200/fff200.png) | Yellow | `#fff200` | C:0 M:5 Y:100 K:0 |

Always ensure sufficient contrast between text and background colors. Yellow on white and black on yellow are not permitted combinations.

## Typography

FLOSSK uses the **Alfabeti** typeface — a custom font designed for the Albanian alphabet.

| Weight | File | Use |
|--------|------|-----|
| DemiBold | [`fonts/Alfabeti-DemiBold.otf`](./fonts/Alfabeti-DemiBold.otf) | All brand communications |
| Regular | [`fonts/Alfabeti-Regular.otf`](./fonts/Alfabeti-Regular.otf) | Supporting text |

**Albanian alphabet (Alfabeti DemiBold):**

```
A B C Ç D DH E Ë F G GJ H I J K L LL M N NJ O P Q R RR S SH T TH U V X XH Y Z ZH
a b c ç d dh e ë f g gj h i j k l ll m n nj o p q r rr s sh t th u v x xh y z zh
```
